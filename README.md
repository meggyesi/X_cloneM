# X_cloneM
## Programrendszerek fejlesztése gyakorlat - Twitter klón kötprog

* Meggyesi Márk(ME8RU1)

## Főbb funkciók:
* Login
* Signup
* Tweet
* Like
* Comment
* Follow
* Delete tweet
* Edit tweet

## Projekt indítás
* npm run + ng serve  - Frontend
* npm run build, npm run start  - Backend

## Használt technológiák:
* Docker - Mongo db
* Angular
* NodeJs, ExpressJs
